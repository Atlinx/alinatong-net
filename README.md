# alinatong.net

Personal professional website hosted at [alinatong.net](https://alinatong.net).

## Development

Install dependencies:

```bash
pnpm install
```

Start the development server on `http://localhost:3000`:

```bash
npm run dev
```

Build the application for production:

```bash
npm run build
```

Locally preview production build:

```bash
npm run preview
```
